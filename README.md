# 筋トレマニアはひかれ合う

## サイト概要
自分のだらしない体にむかっ腹が立ったことはないでしょうか？
筋トレは__食事から始まる__と言っても過言ではありません。
このサイトではトレーニングの記録だけでなく、自身の情報から摂取すべきマクロ栄養素（たんぱく質、脂質、炭水化物）の量を割り出すことができ、それをもとに実際の食事内容を記録していくことができます。
一緒に本当の筋トレをはじめようじゃあないか

### サイトテーマ
トレーニングの記録と、食事の記録を同時にできるサイト
